# ぽこあジェネレーター (Pokoa Generator)

某作品のトップ画面のような、かわいくてポップなロゴ画像を簡単に作成できるWebアプリケーションです。
本ツールは、身内での利用を目的としたお遊びプロジェクトです。

## 機能

- **テキスト入力**: 好きな文字を入力してオリジナルのロゴを作成できます。半角スペースを入力するとその位置で改行されます。
- **フォント選択**: オリジナルの「ぽこぴあフォント」をはじめ、「Potta One」や「Kiwi Maru」など、様々なかわいい日本語オープンソースフォントに対応しています。
- **背景切り替え**: スクリーンショット風の背景、青空、白、黒から、お好みの背景スタイルを選択できます。
- **画像ダウンロード**: 生成した高品質なロゴ画像は、ボタンひとつで保存可能です。

## プライバシーとセキュリティ

- 本ツールは**完全にローカル環境で動作**します。
- 入力された文字列や生成された画像データが外部のサーバーに送信されることは一切ありません。

## 使い方（ローカル実行）

1. 本プロジェクトをローカル環境に用意します。
2. ターミナルで以下のコマンドを実行し、ローカルサーバーを立ち上げます。
   ```bash
   npm install
   npm run dev
   ```
3. プレビュー表示されたURL（例: `http://localhost:...`）をブラウザで開きます。
4. テキストフィールドに好きな文字を入力し、「生成」ボタンを押してロゴを作成します。
5. 気に入った画像ができたら「ダウンロード」ボタンから画像を保存します。

## 使用技術

- HTML / CSS (Vanilla)
- JavaScript (Canvas APIによる画像生成処理)
- Node.js 環境によるローカルサーバー実行
